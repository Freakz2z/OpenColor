# Screenshots

Drop your screenshot files here. The README expects these filenames:

- `main.png` — the main window showing the palette list
- `editor.png` — the color editor with the HSL picker visible
- `picker.png` — the picker window hovering over a screen capture (showing the floating color card)

Recommended size: 1200×800 or similar 3:2 ratio. PNG preferred over JPG to keep text crisp.

To regenerate, run the app, take screenshots, and commit them with `git add screenshots/`.
